## 项目结构
```
coder-algorithm                  # 算法工程目录
├─algorithm                      # 算法
│ ├─maths                        # 数学相关
│ ├─sorts                        # 排序相关
│ └─strings                      # 字符串相关
│
│─.flake8                        # 静态代码检查配置文件
│─.gitignore                     # git忽略文件
│─.pre-commit-config.yaml        # 自动格式化的工具
│─.mypy.ini                      # 静态类型检查配置文件
│─requirements-test.txt          # 依赖文件（测试）
│─requirements.txt               # 依赖文件
│─setup.py                       # 安装程序
└─README.md                      # 简介文件

```
